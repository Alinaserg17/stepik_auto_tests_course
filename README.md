## ⚙️ Как установить и запустить

### 1. Настройка окружения
Скопируйте репозиторий и установите зависимости:

```bash
git clone <ссылка-на-ваш-репозиторий>
cd <папка-проекта>
pip install -r requirements.txt
```

### 2. Запуск тестовых сценариев
Обычная команда `pytest` запускает тесты на **русском** языке (это язык по умолчанию).

**Стандартный прогон:**
```bash
pytest -v test_product_page.py
```

**Только тесты, требующие проверки преподавателем (маркер `need_review`):**
```bash
pytest -v -m need_review test_product_page.py
```

**Запуск с конкретным языком (например, английский):**
```bash
pytest -v --language=en test_product_page.py
```

## 📂 Как устроен проект

* `pages/` — описание всех страниц через классы (BasePage, LoginPage, ProductPage и т.д.).
* `conftest.py` — настройки фикстур, запуск браузера и обработка аргументов из командной строки.
* `pytest.ini` — пользовательские маркеры (`login_guest`, `need_review`).
* `tests/` (или корневые файлы) — сами тест-кейсы.
